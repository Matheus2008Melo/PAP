3. Material e Métodos

Esta secção descreve detalhadamente os recursos técnicos e as metodologias aplicadas no desenvolvimento do projeto WeAreSchool. A escolha rigorosa de cada ferramenta, quer ao nível de hardware como de engenharia de software, foi fundamental para garantir a escalabilidade técnica, a segurança dos dados escolares e o sucesso global na concretização da plataforma educativa.

3.1. Recursos Técnicos

Os recursos de hardware e software foram selecionados de forma a criar um ambiente de desenvolvimento local de alta performance, capaz de suportar as tecnologias web mais modernas e lidar eficientemente com a forte carga de processamento exigida pelas lógicas de back-end.

3.1.1. Recursos de Hardware

O desenvolvimento ininterrupto da plataforma WeAreSchool foi realizado num computador pessoal de elevada performance. Esta máquina assegurou a fluidez total durante a compilação extensiva de assets de front-end, a execução contínua de múltiplos serviços através de servidores locais, e suportou sem quebras a constante interação dinâmica com modelos linguísticos e Agentes de IA auxiliares durante os processos de testes e refatorização de código.

3.1.2. Recursos de Software

A stack de software foi criteriosamente escolhida com base em pilares inegociáveis como a modernidade da sintaxe, a comunidade de suporte ativa, a segurança de rotas e a eficiência exigida a uma aplicação escolar.

A espinha dorsal do projeto WeAreSchool assenta nas seguintes tecnologias:
Linguagem de Servidor: PHP 8+, garantindo processamento rápido e seguro da lógica de negócio.
Framework Back-end: Laravel 11.x, que providenciou a estrutura MVC, a encriptação nativa e o Eloquent ORM.
Sistema de Base de Dados: SQLite (para fase de desenvolvimento rápido) e preparo estrutural para MySQL em produção estruturada, permitindo lidar com interações relacionais complexas.
Reatividade Front-end (Sem Reload): Livewire 3 e Alpine.js, substituindo as pesadas Single Page Applications baseadas puramente em JavaScript e garantindo o funcionamento imediato dos dropdowns dinâmicos e aprovações rápidas.
Motor de Estilização Visual: Tailwind CSS, adotando metodologias de utility-class para garantir um design de excelência em ecrãs móveis (tablets, smartphones) e computadores desktop.
IDE e Assistência IA: Visual Studio Code equipado com poderosos motores de inferência via IA (Cursor/Agentes) pautando o projeto pela metodologia de Desenvolvimento 2.0.
